Dashboard Assignment - TRI Page View Analysis
Analyzing if Users Visit More Pages Before Converting
Ashley Lee
  • TRI Page View - Overview
  • TRI Page View - Analysis
Story

This dashboard examines whether users who view more pages are more likely to begin the booking process. The analysis uses a session-level dataset derived from GA4 events. Conversion is defined as a session that includes a checkout_started event.

Key Insights

  • 60.1% of sessions end after just 1 page.

  • Sessions with 1-3 page views convert at less than 0.5%.

  • Conversion rises to 2.71% at 4-6 pages and 9.65% at 7-15 pages.

  • Highly engaged sessions with 16-50 page views convert at 28.57%.

  • These results suggest a meaningful engagement threshold at 4+ page views.

Conclusion

This analysis shows a clear relationship between page depth and booking intent. Users who browse more deeply through the TRI website are much more likely to begin the checkout process. The largest opportunity for improvement may be increasing early-session engagement so that more users move beyond the first few pages.
